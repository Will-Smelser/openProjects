<static> mioPagination.page.add(weight, dispName, name, eventParams, file, params)
Adds pages to the pagination array Run orderPages() after this if you add after drawing the pagination
Parameters:
{Integer} weight
Where you would like the page to show up in the page que
{String} dispName
The name to be displayed in page navigation
{String} name
A unique name used to access information about the page through built in methods and system calls
{Array} eventParams
An array of events associated with the page...see this._eventParams for list of events
{String} file
The file to be loaded on the page click
-file.file string Path to file to be loaded, either http, or relative and absolute paths
-file.type string The way to load the file, either text or iframe
{Object} params
json Optional paramters which most are defined in info.defaults
-params.clicked bool whether the page has been assumed to be clicked once already
-params.seperate html string of html for the page seperator(default is ">") before the page
-params.allowBackNav bool wether this page can be navigated to again (allows only 1 onclick to run)
-params.sequential bool wether this page must be in sequence...all pages prior to this page must be clicked to view this page
-params.ajaxFailMsg html string of html to be displayed in content area on an AJAX failure (404 or timeout)
